# 팀 대항전 운영 기준 (2025-12-13)

## 일정·참여
- 시즌 길이: 2일 배틀 (KST 기준 시작/종료)
- 팀 선택: 시즌 시작 후 2시간 동안만 노출, 이후 팀 변경 불가
- 배정 방식: "미스터리 팀" 단일 버튼 → 인원 적은 팀으로 자동 배정(무작위+밸런스)
- API: `/api/team-battle/teams/auto-assign` (노출 창 내에서만 허용), 현재 팀 조회 `/api/team-battle/teams/me`
- 운영자 강제 참여: `/admin/api/team-battle/teams/force-join` (선택 창 무시, 운영자 전용)

## 포인트 산정
- 소스: 게임 플레이 횟수만 집계 (승/패 무관)
- 적립: 1회 플레이당 팀 포인트 10점
- 일일 상한: 500점/인/일 (즉 50회까지 반영), 초과분 미반영
- 적립 조건: 당일 이용(외부 랭킹 데이터) 충족 시에만 `GAME_PLAY` 포인트 적립
- 반영 시점: 플레이 즉시 적립, 리더보드 1~5분 내 업데이트

## 진행 지표 (UI 노출)
- 팀 순위 및 격차
- 내 팀 포인트 총합
- 내 기여도(오늘/누적 플레이 수→포인트 환산)
- 시즌 종료까지 남은 시간 (카운트다운)

## 보상 정책
- 팀 순위 보상 (시즌 종료 시 확정)
  - 1위 팀: 팀원 전원 쿠폰 3만 (수동 지급)
  - 2위 팀: 팀원 전원 포인트 100 (자동 지급 가능)
- 개인 기여 보상 (팀 내)
  - TOP3: 쿠폰 1만 (수동 지급)
  - 최소 기여 기준: 시즌 누적 20회 이상 플레이한 인원만 모든 보상 대상
- 자동 vs 수동
  - 포인트형(2위 팀 보상)은 시스템 자동 지급 가능
  - 쿠폰형(1위 팀, TOP3)은 운영자 수동 지급

## 기타 메모
- 시간대: 모든 기준은 Asia/Seoul (KST)
- 팀 배정 이후 변경 불가
- 포인트 소스, 상한, 보상 금액 변경 시 본 문서 갱신 필요

## 비공개 자동 밸런싱 (입금/플레이 기반)
- 기준 일자: 전일(KST) 데이터를 사용해 시즌 시작 전 배정
- 지표: 전일 입금액, 전일 플레이 횟수 (external_ranking_data 일일 증분 기반)
- 전처리: 결측 0 처리, p95 클리핑 후 0~1 정규화, 극단치 로그/상한으로 완화
- 점수: score = 0.6·입금정규화 + 0.4·플레이정규화 (가중치는 운영자 조정 가능)
- 배정: 점수 내림차순 → 팀 총점이 낮은 쪽에 순차 배정, 동점 구간은 셔플(seed 고정)로 패턴 노출 방지
- 적용: `/admin/api/team-battle/teams/auto-balance` (apply=true 시 즉시 반영, 팀 2개 필수)
- 노출: 사용자 UI/API에는 배정 로직, 점수, seed를 노출하지 않음
