# 어드민 전면 교체 체크리스트

참고 문서: 
- frontend API 계약: docs/frontend_api_contract_freeze.md
- 디자인 가이드: docs/admin_page_design_guide.md
- 회원/데이터 페이지 가이드: docs/admin_member_data_page_guide.md
- 디자인 레퍼런스 이미지: 대시보드 스크린샷(첨부)

## 0) 준비
- [ ] 로컬/CI에서 프론트 빌드 확인 (npm run build) 후 시작
- [ ] API 스펙 재확인: auth, member, ticket, ranking, vault, season 등 계약이 freeze 문서와 일치하는지

## 1) 공통 레이아웃/테마 적용
- [ ] `Layout.tsx`를 디자인 가이드(다크 팔레트, 사이드바/헤더)로 교체
- [ ] `Sidebar.tsx` 아이콘/섹션/토글 구조 적용, 모바일 오버레이 포함
- [ ] `Header.tsx` 알림/유저 아바타/토글 버튼 적용
- [ ] 글로벌 폰트 `Noto Sans KR`, 색상 토큰 반영, 기본 배경/테두리/포커스 스타일 일괄 적용
- [ ] 공통 버튼/입력/배지/토스트/모달/탭 컴포넌트 스토리(or 샌드박스)에서 시각 확인

## 2) 대시보드 페이지 교체
- [ ] 메인 카드(활성 사용자/게임 참여/티켓 사용량/평균 체류 시간) 스타일 적용
- [ ] 그리드 카드(시즌 설정, 기능 일정, 룰렛 설정, 주사위 설정 등) UI를 디자인 가이드와 이미지에 맞춰 교체
- [ ] 애니메이션 없는 정적 배너/CTA 확인, 반응형(2~4열) 테스트

## 2-1) 사이드바 탭 구조 점검
- [ ] 게임 관리: 시즌설정 / 기능일정 / 룰렛설정 / 주사위설정 / 복권설정
- [ ] 회원 관리: 신규회원판정 / 회원관리 / 팀배틀관리
- [ ] 티켓 관리: 티켓지급 / 티켓기록·회수
- [ ] 데이터 관리: 랭킹입력 / 사용자분류 / 설문조사
- [ ] 아이콘/라벨/순서가 디자인 레퍼런스 이미지와 일치하는지 확인

## 3) 회원관리(MemberCRUD)
- [ ] 데이터 모델/테이블/인라인 편집/페이지네이션/추가 폼을 가이드대로 구현
- [ ] 정렬/검색/필터/토스트/알림 상태 동작 확인
- [ ] ARIA 및 키보드 접근(Enter/Space) 확인

## 4) 티켓기록/회수(TicketLogsPage)
- [ ] 탭 네비게이션(지갑/플레이로그/코인원장/회수) 구현
- [ ] 선택/확장 행, 고급 필터, 데이터 시각화(라인차트) 연결
- [ ] CSV 내보내기, 그룹화/집계(useMemo) 검증

## 5) 랭킹입력(RankingInput)
- [ ] 테이블 정렬/검색/필터, 인라인 편집/신규행 하이라이트, 일괄 편집 모달 구현
- [ ] 페이지네이션과 선택/토글 모두 확인

## 6) 공통 UX 품질
- [ ] 로딩/에러/빈 상태 UI 확인
- [ ] 토스트/알림 자동 숨김 동작 확인
- [ ] 드롭다운/모달 외부 클릭 닫힘 동작 확인
- [ ] 반응형(모바일/태블릿/데스크톱) 레이아웃 테스트
- [ ] 색상 대비/포커스 링 확인(접근성)

## 7) 성능 최적화
- [ ] 큰 테이블에 react-window 등 가상화 적용 (필요 시)
- [ ] 검색/필터 입력에 디바운스 적용
- [ ] 메모이제이션(useMemo/useCallback) 확인
- [ ] 지연 로딩(lazy/Suspense) 필요한 무거운 뷰에 적용

## 8) API 연동 점검
- [ ] 모든 뷰가 frontend_api_contract_freeze.md의 스키마로 호출/표시 되는지 점검
- [ ] 에러 메시지/빈 결과/권한 오류 시 UI 피드백 확인

## 9) 테스트 및 품질 보증
- [ ] 단위/통합 테스트 추가 또는 업데이트 (테이블 로직, 필터, 정렬, 모달 동작)
- [ ] E2E 시나리오: 로그인 → 대시보드 → 회원관리 CRUD → 티켓 로그 필터/CSV → 랭킹 입력/일괄 편집 → 로그아웃
- [ ] CI 빌드 통과 확인

## 10) 배포 전 점검
- [ ] 환경 변수/엔드포인트(스테이지/프로덕션) 확인
- [ ] 번들 사이즈/성능 예열, 주요 페이지 LCP/TTI 수치 체크
- [ ] 변경사항 릴리스 노트 작성
