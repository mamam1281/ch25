# 2026-01-02 개발 로그: 기술 부채 해결 및 신규 회원 프로모션 품질 개선

## 개요
관리자 패널의 회원 삭제 에러 수정과 신규 회원 프로모션 모달의 프론트 안정성 강화를 진행했습니다. 또한 원활한 협업을 위해 깃 개발 환경을 재설정했습니다.

## 상세 내용

### 1. 회원 삭제 기능 (Admin Panel)
- **에러 발생**: 회원 삭제 시 연관 데이터(Vault Status)의 CASCADE 설정 누락으로 인한 500 에러 발생.
- **조치**: 
  - `vault_status` 테이블에 `user_id`, `program_id` CASCADE 외래키 설정을 추가하는 Alembic 마이그레이션 적용.
  - `AdminUserService` 로직 정비 및 레거시 코드 정리.

### 2. 신규 회원 프로모션 모달 (Frontend)
- **에러 발생**: 텔레그램 채널 구독 인증 시 간헐적인 프론트 멈춤 및 데이터 비동기 처리 중 모달 증발 현상.
- **조치**:
  - **예외 처리**: 미션 액션 호출 시 `try...catch` 및 로딩 상태(`isVerifying`)를 추가하여 중복 클릭과 튕김 현상 방지.
  - **데이터 유지**: `useRef`를 활용해 API 재조회(polling) 중에도 기존 미션 데이터를 유지하여 모달이 사라지지 않도록 개선.
  - **사용성 개선**: 게임 플레이 미션 클릭 시 토스트 알림과 함께 해당 페이지로 이동 기능 추가.
  - **채널 유지**: "씨씨지민"과 "씨씨공식" 두 채널의 목적에 맞게 개별 링크 체계 유지.

### 3. 개발 인프라 및 운영
- **깃 설정**: 로컬 브랜치의 업스트림을 개인 포크 저장소(`mamam1281/ch25`)로 고정하여 배포 프로세스 효율화.
- **서버 반영**: 수동 복사 및 컨테이너 재시작을 통한 실서버 패치 완료 (사용자 삭제 기능 등).
