## 2. 이벤트 항목별 상세 기록 및 평가

### A. 고액 주사위 한정 이벤트
*   **내용**: 승리 시 금고 +2,000원 적립, 패배 시 -2,000원 차감. (시간 한정)
*   **평가 (Economy Alignment)**:
    *   **리스크**: 기존 표준(승 +200 / 패 -50) 대비 **10~40배** 높은 고위험-고수익 구조임.
    *   **심리 효과**: "돈 넣고 돈 먹기" 프레임을 강화하여 단기 도파민과 체류 시간을 폭발적으로 늘릴 수 있음.
    *   **주의사항**: 패배 시 차감액(-2,000)이 크므로, 금고 잔액이 부족한 유저에게 
                    "미수금(차감)"이 발생할 때의 이탈 리스크를 관리해야 함.
    *   **의견**: 단기 이벤트로 적절하지만, 장기 운영 시 금고 자산 인플레이션의 
                주범이 될 수 있으므로 세션 시간을 엄격히 제한(예: 30분~60분)해야 함.

### B. 충전 연동 키(Key) 할인/지급 이벤트
*   **내용**: 단일 입금 금액에 따른 키 지급. (20만 -> 골드 1, 50만 -> 다이아 1, 100만 -> 다이아 3)
*   **평가 (Retention Cycle)**:
    *   **전환 유도**: 외부 CC 충전을 직접적으로 앱 내 고가치 재화(Key)로 연결하는 훌륭한 전환 트리거임.
    *   **신규 유저 유입**: "고액 룰렛"에 대한 접근성을 높여 Gold/Diamond 룰렛의 
                            금고 누적 경험을 빠르게 체감하게 함.
    *   **운영 가이드**: 해당 보상은 `RewardService` 자동화보다는 현재처럼 **"충전 후 지민이에게 알림"** 식의 수동 검증 방식을 유지하여 부정 수급(단일 금액 미준수 등)을 방어하는 것이 현명함.

### C. 입금액별 티켓 패키지 (코드지갑 게임 오픈)
*   **내용**: 1만(주사위 2), 10만(주사위 4), 20만(주사위 5+복권 1), 50만(티켓풀 10).
*   **평가 (Fuel Supply)**:
    *   **연료 공급**: 소액러부터 고액러까지 촘촘하게 티켓을 공급하여 "플레이가 끊기지 않게" 만드는 리텐션의 기본에 충실함.
    *   **효율성**: 50만 입금 시 "티켓 풀 10개" 같은 대량 지급은 유저를 앱 내에 오래 묶어두는 효과가 있음.

### D. 2026년 첫 팀 배틀
*   **내용**: 점수 300점 이상 팀원 중 상위 팀에게 최대 30만원 보상. (횟수 기반 집계)
*   **평가 (Social & Competition)**:
    *   **안전장치 (Anti-Abuse)**: "버스 금지(300점 이상)" 조건과 "플레이 횟수로만 집계"하는 룰은 매우 훌륭함. 돈을 많이 쓰는 유저뿐 아니라 소액으로도 열심히 플레이하는 "진성 유저"에게 역전 기회를 주어 소속감을 강화함.
    *   **리텐션 효과**: 15일이라는 긴 기간 동안 팀의 승리를 위해 매일 접속하게 만드는 강력한 중기 리텐션 요소임.

### E. X2배 적립 찬스
*   **내용**: 특정 시간에 게임 승리 시 금고 적립 2배 (Multiplier 적용).
*   **평가 (Peak-time Nudge)**:
    *   **피크타임 밀집**: 특정 시간대에 유저를 몰아넣어 실시간 당첨 피드(To-Be)의 효과를 극대화할 수 있는 사전 정지 작업임.
    *   **경제 영향**: `VAULT_ACCRUAL_MULTIPLIER=2.0` 설정을 통해 시스템적으로 자동 대응 가능한 이벤트임.

## 3. 종합 평가 및 제언 (CC Retention Polaris)

### 👍 긍정적 측면
1.  **확실한 보상 체감**: +2,000원/Key 지급 등 유저가 "확실히 벌어간다"는 느낌을 받도록 설계됨.
2.  **공평한 경쟁**: 팀 배틀에서 플레이 횟수 기반 집계를 택하여 하드 유저와 일반 유저의 밸런스를 잘 잡음.
3.  **외부 충전 연결**: 모든 이벤트가 단일 입금액과 연동되어 실질적인 매출(CC 충전)로 직결되도록 짜여 있음.

### ⚠️ 주의 및 개선 제안
1.  **금고 패널티 관리**: 주사위 패배 시 -2,000원은 유저에게 큰 타격일 수 있음. 금고가 0원 이하로 내려갈 경우, `TRIAL_TOKEN` 루프로 자연스럽게 넘어가게 하거나 "부활 이벤트" 카피를 미리 준비해야 함.
2.  **수동 정산 리스크**: "지민이에게 알려주세요" 방식은 유합 폭주 시 CS 병목이 발생할 수 있음. 향후 어드민에서 **"입금액 확인 후 즉시 키 지급"** 버튼 하나로 처리할 수 있는 기능을 강화해야 함.
3.  **이벤트 피로도**: 고액 주사위와 2배 적립이 동시에 일어날 경우 경제 인플레이션 속도가 매우 빠를 수 있음. 주간 단위로 총 지급액(Payout)을 모니터링하여 배율을 조정할 필요가 있음.

---

**결론**: 현재 운영안은 **'강력한 한방 보상'**과 **'성실한 플레이 보상'**이 잘 섞여 있어 단기 모객과 중기 리텐션 모두에 효과적일 것으로 판단됨. 다만, 수동 정산의 피로도를 낮추기 위한 시스템 보강이 병행되어야 함.
